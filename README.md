# 팀블로그 프로젝트

## 개요
Vue.js및 다양한 API를 이용하고, PWA 개념을 적용한 웹 모바일 페이지

## 프로젝트 기간
2019.07.01 ~ 2019.08.16

## 개발 환경
- Python
- Vue.js
- Firebase
- Vuetify
- AWS

## 담당 역할
프론트엔드 엔지니어 (기여도 20%)

## 담당 기능
- 오프라인 기능 service worker 구현
- 새로운 글 업데이트 시 Firebase API를 활용한 push 알림
- 외부 날씨API를 활용하여 현재 위치의 날씨 정보 제공
